React Guide

Getting Started with React

The following instructions will guide you through starting a new PX Blue project using React. Alternatively, you can also start your project by downloading one of our code samples from Github (the Login or Navigation examples are good projects to start from).

Using the PX Blue CLI

The fastest way to start a new PX Blue project is to use the PX Blue CLI. The CLI will automatically create a starter project for you (using Create React App) with all of the necessary PX Blue dependencies pre-configured. To start a React project:

npx -p @pxblue/cli pxb new react

You will be asked to provide a name for your project, choose whether you would like to use TypeScript or JavaScript, and for TypeScript projects, you'll be asked if you want to use the PX Blue eslint and prettier configurations files.

The PX Blue CLI will automatically include:

  • Material-UI Component Library and Icons (@material-ui/core / @material-ui/icons)
  • PX Blue Component Library (@pxblue/react-components)
  • PX Blue Themes (@pxblue/react-themes)
  • PX Blue Icons (@pxblue/icons-mui)
  • PX Blue Colors (@pxblue/colors)
  • Open Sans web font (typeface-open-sans)
  • PX Blue ESLint and Prettier configurations, if selected (@pxblue/eslint-config / @pxblue/prettier-config)
  • IE 11 Polyfills (react-app-polyfill)
  • Enzyme testing library (enzyme / enzyme-adapter-react-16)

For detailed usage instructions, refer to the PX Blue CLI documentation.

Using an Existing Project

If you have an existing project and would like to integrate PX Blue, you can follow the manual integration instructions below. If you have not already started your project, we recommend that you scaffold your project with the PX Blue CLI.

Manual PX Blue Integration

Before taking the following steps, you'll want to make sure that you have an existing application. If you used the PX Blue CLI to start your project, these steps have already been done for you automatically.

Including the Component Libraries

Material UI

React projects should make use of the Material-UI component library. Our PX Blue themes are designed to work with these components. You'll also want to install the icons.

npm install --save @material-ui/core @material-ui/icons
// or
yarn add @material-ui/core @material-ui/icons

For detailed documentation about the various components available in Material UI, as well as sample code and API reference, refer to the Material UI Documentation.

PX Blue Components

Additionally, we have a supplemental library of components designed specifically for PX Blue applications. If you wish to use these components, you can install them by running:

npm install --save @pxblue/react-components
// or
yarn add @pxblue/react-components

Adding PX Blue Themes

By default, your application will be themed with the Google Material theme. To use the PX Blue theme, first install it:

npm install --save @pxblue/react-themes
// or
yarn add @pxblue/react-themes

Installing the theme package will automatically include the Open Sans web font package as well as the PX Blue colors package.

Applying the Theme

Import the theme files, Material UI theme provider, and Open Sans font into your application, e.g. in App.js:

import { ThemeProvider, createMuiTheme } from '@material-ui/core/styles';
import CssBaseline from '@material-ui/core/CssBaseline';
import * as PXBThemes from '@pxblue/react-themes';
require('typeface-open-sans');

To add the theme to your whole application, you would simply add a <ThemeProvider> around your top-level component and pass it the theme you want to use, e.g.:

<ThemeProvider theme={createMuiTheme(PXBThemes.blue)}>
    <CssBaseline/>
    <MyApp/>
</MuiThemeProvider>

You can read about React Theming to learn more.

Additional Information

Code Formatting/Style

PX Blue provides packages for code standards and style using ESLint and Prettier. These packages are used internally by the PX Blue team, but they can also be used in your projects. You can find instructions for adding these packages to your project on GitHub.

NOTE: the ESLint package is only available for TypeScript projects.

Browser Support

React and Material UI will work with any modern browser. For specific details, you can view their official support pages:

  • React
  • Material UI

The latest Create React App CLI does not include support for IE11 by default. If you need to support this browser (not recommended), you can configure your project to support it manually (Contact us for assistance) or use the PX Blue CLI to scaffold your project.
